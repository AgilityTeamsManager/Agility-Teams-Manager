A propos de Agility Teams Manager...
====================================

**Agility Teams Manager** est un logiciel gratuit et open-source qui permet de générer des classements par équipes pour les concours d'agility.

Il dispose d'une interface claire et simple d'utilisation, mais est rempli de fonctionnalités pour l'utilisateur avancé.

Il est configurable à volonté.

Où aller ?
==========

Si vous êtes un développeur, vous pouvez lire :doc:`developpement`.
